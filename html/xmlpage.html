SAMPLE WEB PAGE Welcome to one of the worlds first WWW Home pages written entirely in XML. This Web page has been constructed using: basic XML code (derived from HTML) a CSS style sheet to get the page to display in Microsoft Internet Explorer a separate CSS style sheet to get the page to display in Netscape Communicator another CSS style sheet to get the page to display in Netscape Mozilla (a September 1998 build) variations of the XML code embedded as pseudo-HTML code inside a standard HTML web page Note that all of this code is experimental so dont be disappointed if something doesnt quite work the way you think it should.
